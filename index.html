VANILLA SLAB

A non-jquery, modified clone of the excellent jquery slabtext plugin.

Default usage:

Add a class of 'js-vanilla-slab' to a header on your page. Then initiate like this: 

      VanillaSlab.init();

With Options:

Add your own class and initialize with options:

      VanillaSlab.init({
        selector: '.my-class',
        maxWordsPerLine: 4,
        minWordsPerLine: 2,
        ...
      });
